# Vibe UI 🎨

Premium, animated, glassmorphism React components for modern developers.
Vibe UI is not a component library you install via NPM. It's a collection of beautiful, accessible components that you copy and paste directly into your Next.js/React apps.

## 🌟 View the Documentation

This repository contains the full interactive documentation website (built with Next.js 15).

To run the documentation locally and view the components:

```bash
git clone https://github.com/nugrahanastya/vibe-ui.git
cd vibe-ui
npm install
npm run dev
```

Then open `http://localhost:3000` in your browser.

## 🛠️ Tech Stack
- **Next.js 15** (App Router)
- **Tailwind CSS v4** (Styling)
- **Framer Motion** (Animations)
- **Lucide React** (Icons)

## 📦 How to use the components
Browse to `http://localhost:3000`, preview the components, and click the **Copy Code** button. 

Or, if you prefer the raw files, look inside the `src/registry/components/` folder.
